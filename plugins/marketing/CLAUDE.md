# Marketing Plugin

Growth and content roles for SaaS and software products.

## Agents

Six roles, each with a clear mandate. Invoke via the Agent tool.

| Agent | One job |
|-------|---------|
| **strategist** | Positioning, competitive intel, pricing, marketing frameworks |
| **content-writer** | Write and edit copy for any surface |
| **seo-engineer** | Technical SEO, schema markup, programmatic pages |
| **cro-analyst** | Optimize conversion on any page or flow |
| **growth-manager** | Acquisition channels — launches, ads, referral, free tools |
| **analytics-engineer** | Tracking setup, measurement, experiment design |

## Skills (25)

Skills are domain knowledge loaded by agents. You don't invoke them directly — the agents read the relevant skill when they need it.

### Strategist
- `product-marketing-context` — foundational positioning document
- `marketing-psychology` — 70+ mental models and cognitive biases
- `pricing-strategy` — value metrics, tier design, packaging
- `competitor-alternatives` — comparison and alternative pages
- `marketing-ideas` — 139 categorized marketing tactics

### Content Writer
- `copywriting` — page copy, headlines, CTAs
- `copy-editing` — seven-sweep editorial methodology
- `content-strategy` — topic clusters, buyer journey, content calendar
- `social-content` — platform-specific post creation
- `email-sequence` — drip campaigns, nurture flows, lifecycle emails

### SEO Engineer
- `seo-audit` — technical SEO audit methodology
- `schema-markup` — JSON-LD structured data
- `programmatic-seo` — template-driven pages at scale

### CRO Analyst
- `page-cro` — landing page, homepage, pricing page optimization
- `signup-flow-cro` — registration and trial activation
- `form-cro` — lead capture, contact, demo request forms
- `popup-cro` — exit intent, scroll triggers, email capture
- `onboarding-cro` — post-signup activation and first-run experience
- `paywall-upgrade-cro` — in-app paywalls and upgrade flows

### Growth Manager
- `launch-strategy` — product launches, Product Hunt, phased rollouts
- `paid-ads` — Google, Meta, LinkedIn, TikTok campaigns
- `referral-program` — viral loops, incentive design, affiliate mechanics
- `free-tool-strategy` — engineering-as-marketing tools

### Analytics Engineer
- `analytics-tracking` — GA4, event tracking, UTM strategy
- `ab-test-setup` — experiment design, sample size, hypothesis frameworks
